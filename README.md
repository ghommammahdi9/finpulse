# FinPulse — Analyseur intelligent de relevés bancaires
Application web d analyse financiere personnelle combinant Data Engineering et IA.
## Fonctionnalites
- Extraction automatique de transactions depuis PDF bancaires
- Categorisation intelligente via LLM Anthropic Claude
- Detection d anomalies statistique
- Dashboard interactif avec visualisations Plotly
## Stack
Python, Pandas, pdfplumber, SQLite, Anthropic API, Streamlit, Plotly
## Lancement
streamlit run src/app.py
